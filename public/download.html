How to install

1

Tap Install GroundLink above to download the app file.

2

If your browser warns about the file type, choose Download anyway — it's safe, it's just not from the Play Store.

3

Open the downloaded file and tap Install. The first time, Android asks you to allow installs from this app — turn it on, then tap Install.

4

Open GroundLink and you're in. It keeps itself up to date automatically — no need to reinstall.

📱 On an iPhone? This installable app is Android-only. You can still use GroundLink in Safari — open the link your friend shared, then tap Share → Add to Home Screen.
Already have it installed? Just reopen the app — it updates itself.
Trouble installing? Make sure you're on Android and try again.
